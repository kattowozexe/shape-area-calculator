# shape-area-calculator

Overview
The Shape Area Calculator is a Python-based command-line application that calculates the area of basic geometric shapes, including square, rectangle, triangle, and circle.
This project was developed to practice fundamental programming concepts such as control flow, user input handling, and mathematical computation.

Features
Menu-driven interface for shape selection
Continuous execution using a loop until user exits
Area calculation for:
Square
Rectangle
Triangle
Circle
Basic input validation for invalid menu choices

Concepts Practiced
while loop for repeated execution
if–elif–else conditional branching
User input handling with input()
Arithmetic operations and exponentiation (**)
Program termination using break

Program Structure
Display menu options
Prompt user to select a shape
Request necessary dimensions
Apply corresponding geometric formula
Output calculated area
Repeat until user chooses to quit

Example Formulas Used

Square:
area = side²

Rectangle:
area = length × width

Triangle:
area = (base × height) / 2

Circle:
area = 3.14 × radius²

Lesson
Through this project, I strengthened my understanding of translating mathematical formulas into programmable logic. I also learned to structure user interaction clearly and balance functionality with simplicity in code design.
